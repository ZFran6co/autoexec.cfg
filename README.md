$ Installation Guide

1. Download or create the `autoexec.cfg` file.
2. Place it in the CS2 configuration folder:
   **Path**: `C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg`  
   *(If CS2 is installed elsewhere, navigate to that location instead)*.
3. `exec autoexec.cfg` in the console.
